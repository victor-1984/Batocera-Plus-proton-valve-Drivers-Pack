# Drivers Pack Para "proton-valve" De  Batocera Plus

Este pack contiene una recopilación de "Drivers" para "proton-valve" para una mejor compatibilidad para los juegos de "Windows"

Una vez descargado el "Pack" se recomienda ejecutar en el orden que se indica

Primero (1º Wine Start.plus)

Segundo (2º Wine Drivers.plus)

Al ejecuta el primer "script" configura los "Cores de Wine" y luego descarga e instala 3 driver adicionales automáticamente desde internet que son necesarios para una gran mayoría de juegos.

Luego al ejecutar el seguro "script" es automático y no necesita descargar nada de internet ya que todos los drivers estan descargados en la misma carpeta.

Recomendacion.

cuando ejecuten el segundo script y cuando la instalación llegue a una ventana de "DirectX 11" y pregunte donde descomprimir para instalar, el recuadro de la ruta escribe a mano /tmp/ y luego preciona en "aceptar o continuar"
